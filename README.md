
Mac専用

事前にXcodeとAndroid Studioをインストールしておく

cocoapodsとかJava11とかもインストールしておく

### Xcodeプロジェクトの起動

```bash
$ npx cap open ios
```


### iOSでのビルド + エミュレータの起動

```bash
$ yarn dev:ios
```


### Android Studioプロジェクトの起動

```bash
$ npx cap open android
```


### Androidでのビルド + エミュレータの起動

```bash
$ yarn dev:android
```
